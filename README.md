# Keybinds
| Key | Description |
| - | - |
| `super`+`enter` | Opens a new terminal (`foot`) |
| `super`+`D` | Opens the application menu (`wofi`) |
| `super`+`shift`+`Q` | Closes current window |
| `super`+`shift`+`N` | Opens the notification center (`swaync`) |
